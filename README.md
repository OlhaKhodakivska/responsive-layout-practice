# The Art of Drinking Coffee ☕

Responsive blog layout for coffee enthusiasts. Created as a practice for **Mobile First** design, **CSS Grid**, and **Media Queries**.

## 🚀 Demo
[Check out the live site here!](https://olhakhodakivska.github.io/responsive-layout-practice/)

## ✨ Features
- **Mobile First Approach**: Optimized for small screens first.
- **Responsive Header**: Title stays in one line on desktop and centers on mobile.
- **Dynamic Grid**: Sidebar moves below the content on mobile devices.
- **Interactive Form**: Centered subscription form for desktop users.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Grid & Flexbox)
- Media Queries
