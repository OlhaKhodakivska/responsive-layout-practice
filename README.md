# ✨ Features

- 📱 **Mobile First Approach** — layout optimized starting from small screens
- 🧱 **CSS Grid Layout** — flexible structure for content and sidebar
- 🔁 **Responsive Design** — adapts across mobile, tablet, and desktop
- 📌 **Responsive Header** — centered on mobile, single-line on desktop
- 🔄 **Adaptive Sidebar** — moves below content on smaller screens
- 📨 **Centered Subscription Form** — optimized desktop presentation
- 🎯 Clean and minimal UI focused on readability
